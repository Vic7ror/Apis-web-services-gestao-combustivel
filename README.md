# Apis-web-services-gestao-combustivel

Api de controle de consumo de combustível de veículos e seus usuários, possuindo autenticação e níveis de autorização

--------------------------------------------

- Aplicativos usados: Visual Studio
- Swagger para documentação
- Postman para testes
- Lenguage C#, ASP_NET

--------------------------------------------
